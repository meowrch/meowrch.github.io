Эффективное использование горячих клавиш может значительно повысить вашу продуктивность при работе с повседневными приложениями. 

В этом руководстве рассмотрены основные сочетания клавиш для наиболее частоиспользуемых компонентов системы:
- [Браузера Firefox](#firefox)
- [Терминала Kitty](#kitty)
- [Файлового менеджера Nemo](#nemo)

## Firefox

Firefox предлагает более 140 различных горячих клавиш для ускорения работы с браузером. Рассмотрим наиболее полезные из них для повседневного использования.

### Управление вкладками

Быстрое управление вкладками позволяет держать рабочее пространство организованным и не тратить время на поиск нужной информации.

| Действие | Комбинация клавиш |
|----------|-------------------|
| Открыть новую вкладку | `Ctrl + T` |
| Закрыть текущую вкладку | `Ctrl + W` |
| Восстановить закрытую вкладку | `Ctrl + Shift + T` |
| Переключиться на следующую вкладку | `Ctrl + Tab` или `Ctrl + Page Down` |
| Переключиться на предыдущую вкладку | `Ctrl + Shift + Tab` или `Ctrl + Page Up` |
| Перейти к вкладке по номеру (1-8) | `Alt + 1` до `Alt + 8` |
| Перейти к последней вкладке | `Alt + 9` |
| Переместить вкладку влево/вправо | `Ctrl + Shift + Page Up/Down` |

<Aside type="tip">
Восстановление закрытых вкладок работает для всех вкладок, закрытых в текущей сессии — вы можете нажимать `Ctrl + Shift + T` несколько раз подряд для восстановления нескольких вкладок.
</Aside>

### Навигация и поиск

| Действие | Комбинация клавиш |
|----------|-------------------|
| Фокус на адресной строке | `Ctrl + L` или `F6` |
| Быстрый поиск в тексте страницы | `Ctrl + F` |
| Найти следующее совпадение | `F3` или `Ctrl + G` |
| Найти предыдущее совпадение | `Shift + F3` или `Ctrl + Shift + G` |
| Вернуться назад | `Alt + Left` или `Backspace` |
| Перейти вперед | `Alt + Right` |
| Перейти на домашнюю страницу | `Alt + Home` |

<Aside type="note">
В адресной строке можно использовать быстрый поиск по конкретным источникам, набрав `@` и выбрав нужный источник (например, `@amazon`, `@bing`, `@history`).
</Aside>

### Закладки и история

| Действие | Комбинация клавиш |
|----------|-------------------|
| Добавить в закладки | `Ctrl + D` |
| Показать все закладки | `Ctrl + Shift + O` |
| Открыть боковую панель закладок | `Ctrl + B` |
| Открыть историю | `Ctrl + H` |

## Kitty

Kitty — это быстрый эмулятор терминала с поддержкой разделения экрана и вкладок. Ваши настройки уже оптимизированы для продуктивной работы.

### Управление вкладками

Вкладки в Kitty позволяют организовать несколько сессий терминала в одном окне.

| Действие | Комбинация клавиш |
|----------|-------------------|
| Открыть новую вкладку | `Ctrl + Shift + T` |
| Закрыть текущую вкладку | `Ctrl + Shift + Q` |
| Следующая вкладка | `Ctrl + Shift + Right` |
| Предыдущая вкладка | `Ctrl + Shift + Left` |
| Перейти к вкладке (следующая/предыдущая) | `Ctrl + Tab` / `Ctrl + Shift + Tab` |


### Разделение экрана

| Действие | Комбинация клавиш |
|----------|-------------------|
| Вертикальное разделение | `Ctrl + Shift + L` |
| Горизонтальное разделение | `Ctrl + Shift + N` |
| Следующее окно | `Ctrl + Shift + ]` |
| Предыдущее окно | `Ctrl + Shift + [` |
| Закрыть текущее окно | `Ctrl + Shift + W` |

## Nemo

Nemo — файловый менеджер по умолчанию в Linux Mint с поддержкой вкладок и множеством горячих клавиш.

### Управление вкладками

| Действие | Комбинация клавиш |
|----------|-------------------|
| Открыть новую вкладку | `Ctrl + T` |
| Закрыть текущую вкладку | `Ctrl + W` |
| Открыть новое окно | `Ctrl + N` |
| Перейти к вкладке по номеру (1-8) | `Alt + 1` до `Alt + 8` |

### Работа с файлами

| Действие | Комбинация клавиш |
|----------|-------------------|
| Открыть терминал в текущей директории | `Ctrl + Enter` |
| Переименовать файл | `F2` |
| Создать новую папку | `Ctrl + Shift + N` |

### Навигация

| Действие | Комбинация клавиш |
|----------|-------------------|
| Перейти на уровень вверх | `Alt + Up` или `Backspace` |
| Вернуться назад | `Alt + Left` |
| Перейти вперед | `Alt + Right` |
| Перейти в домашнюю папку | `Alt + Home` |
| Перейти к адресной строке | `Ctrl + L` |
| Поиск файлов | `Ctrl + F` |
| Показать скрытые файлы | `Ctrl + H` |
