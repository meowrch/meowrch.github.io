**Lutris** — это открытая игровая платформа для Linux, которая упрощает установку и управление играми. Она позволяет запускать игры, разработанные для Windows, а также объединять ваши библиотеки из таких сервисов, как Steam, GOG и Epic Games Store, в одном удобном интерфейсе.

![Lutris](https://upload.wikimedia.org/wikipedia/commons/a/ab/Lutris-Screenshot_%282%29.png)

## Установка

Установка Lutris в Arch Linux выполняется в несколько простых шагов с использованием менеджера пакетов `pacman`.

<Steps>
1.  **Установите Lutris**

    Lutris находится в официальном репозитории `extra`, поэтому его можно установить напрямую [5][8].
    ```
    sudo pacman -Sy lutris
    ```

2.  **Установите зависимости для игр Windows**

    Для запуска большинства игр для Windows вам понадобится Wine. Также рекомендуется установить дополнительные компоненты для лучшей совместимости [3].
    ```
    sudo pacman -Sy wine winetricks
    ```
</Steps>

<Aside type="note" title="Запуск">
После завершения установки вы можете найти Lutris в меню приложений или запустить его из терминала командой `lutris`.
</Aside>

## Использование Lutris

Основной интерфейс Lutris интуитивно понятен. Слева находится панель "Источники" (Sources), где отображаются подключенные сервисы (GOG, Steam и т.д.), а основную часть окна занимает ваша библиотека игр.

### Установка игр через установочные скрипты

Это самый простой и рекомендуемый способ установки игр. Lutris предоставляет готовые скрипты, которые автоматически настраивают всё необходимое для запуска игры.

<Steps>
1.  **Нажмите на иконку `+`** в левом верхнем углу окна Lutris.

2.  Выберите опцию **"Search the Lutris website for installers"**.

3.  В открывшемся окне введите название игры, которую хотите установить. Например, *Overwatch*.

    <Aside type="tip">
    Рекомендуется искать именно игру, а не лаунчер, так как скрипты для конкретных игр могут содержать важные оптимизации.
    </Aside>

4.  Выберите подходящую версию из результатов поиска и нажмите **"Install"**.

5.  Lutris предложит скачать необходимые файлы и компоненты. Просто нажмите **"Continue"** и следуйте инструкциям на экране.
</Steps>

### Подключение сторонних сервисов

Lutris позволяет интегрировать ваши игровые библиотеки из других магазинов, таких как GOG, Humble Bundle и Steam.

1.  На левой панели "Источники" найдите нужный сервис (например, `GOG`).
2.  Нажмите на иконку подключения (персонаж с плюсом) рядом с названием сервиса.
3.  Войдите в свою учетную запись. После успешной авторизации ваши игры из этой библиотеки появятся в Lutris.

### Ручная установка игры

Если для игры нет установочного скрипта, вы можете добавить её вручную, например, если у вас есть установочный `.exe` файл.

<Steps>
1.  Нажмите на иконку **`+`** и выберите **"Add locally installed game"**.

2.  Заполните информацию об игре:
    *   **Name**: Название игры.
    *   **Runner**: Выберите `Wine (Runs Windows games)`.

3.  Перейдите на вкладку **"Game options"**.
    *   **Executable**: Укажите путь к `.exe` файлу игры.

4.  Перейдите на вкладку **"Runner options"**.
    *   **Wine version**: Выберите версию Wine. Обычно подходит последняя системная версия.

5.  Нажмите **"Save"**. Игра появится в вашей библиотеке.
</Steps>

### Управление играми

После установки игра появится в главном окне Lutris.

*   **Для запуска игры** просто выберите её и нажмите **"Play"**.
*   **Для доступа к настройкам** щелкните правой кнопкой мыши по игре и выберите **"Configure"**. Здесь вы можете изменить версию Wine, настроить параметры графики, включить DXVK/VKD3D и многое другое.
*   **Для создания ярлыка** на рабочем столе или в меню приложений щелкните правой кнопкой мыши и выберите **"Create desktop shortcut"** или **"Create application menu shortcut"**.
