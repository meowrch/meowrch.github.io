PortProton — это удобный инструмент на основе Wine и Proton, позволяющий запускать Windows-игры и приложения на Linux. Программа автоматизирует настройку префиксов Wine и обеспечивает изоляцию приложений друг от друга.

## Установка

Установите PortProton через ваш AUR-хелпер:

```
yay -S portproton
```

## Первый запуск

После установки запустите PortProton из меню приложений или через терминал командой `portproton`. При первом запуске программа автоматически загрузит необходимые компоненты Wine/Proton и создаст базовую структуру каталогов.

<Aside type="tip">
  Процесс инициализации может занять 10-30 минут в зависимости от скорости интернета.
</Aside>

## Использование лаунчера

### Главное окно

Интерфейс PortProton состоит из нескольких основных вкладок:

- **Игры** — автоматическая установка популярных лаунчеров (Steam, Epic Games, Battle.net и др.)
- **Установлено** — список установленных игр и приложений
- **НАСТРОЙКИ WINE** — управление префиксами и версиями Wine/Proton
- **Автоустановка** — быстрая установка игровых клиентов

### Автоустановка лаунчеров

<Steps>

1. Откройте PortProton и перейдите на вкладку **Игры**.

2. Выберите нужный лаунчер из списка (Epic Games Store, Battle.net, Origin и др.).

3. Нажмите на кнопку установки выбранного лаунчера.

4. Дождитесь завершения автоматической установки — PortProton самостоятельно скачает и настроит все необходимые компоненты.

5. После установки лаунчер появится в меню приложений системы.

</Steps>

<Aside type="note">
  PortProton автоматически устанавливает необходимые библиотеки DirectX, Visual C++ и .NET Framework для корректной работы лаунчеров.
</Aside>

### Запуск установщиков игр

<Steps>

1. Найдите установочный файл игры (обычно `setup.exe` или `install.exe`) в файловом менеджере.

2. Кликните правой кнопкой мыши по файлу и выберите **«Запустить в PortProton»**.

   Альтернативно: откройте PortProton → нажмите **«Add a new game»** → выберите установочный файл.

3. Создайте новый префикс для игры — рекомендуется использовать отдельный префикс для каждой игры.

   Введите имя префикса (например, название игры).

4. Согласитесь с установкой рекомендуемых библиотек, нажав **«OK»**.

5. Следуйте инструкциям стандартного установщика Windows.

   При выборе папки установки оставьте путь по умолчанию (виртуальный диск C:).

6. После завершения установки создайте ярлык:
   
   - Откройте PortProton → вкладка **«Установлено»**
   - Нажмите **«Создать ярлык»**
   - Выберите префикс и найдите исполняемый файл игры (.exe)
   - Подтвердите создание ярлыка

</Steps>

### Запуск портативных игр

Для игр, не требующих установки:

<Steps>

1. Запустите PortProton из меню приложений.

2. Нажмите **«Add a new game»** (Добавить игру).

3. В файловом менеджере выберите исполняемый файл игры (.exe).

4. PortProton автоматически создаст префикс и запустит игру.

</Steps>

## Управление префиксами

Все префиксы хранятся в домашнем каталоге:

<FileTree>

- home/
  - username/
    - PortProton/
      - prefix/
        - GameName1/
        - GameName2/

</FileTree>

Для настройки существующего префикса:

<Steps>

1. Откройте PortProton и перейдите на вкладку **«НАСТРОЙКИ WINE»**.

2. Выберите нужный префикс из списка.

3. Настройте параметры:
   - Версия Wine/Proton
   - Графический API (DXVK/VKD3D/OpenGL)
   - Разрешение и виртуальный рабочий стол
   - Дополнительные библиотеки

4. Сохраните изменения.

</Steps>

<Aside type="tip">
  Для удаления игры просто удалите папку её префикса из директории `~/PortProton/prefix/`.
</Aside>

## Запуск игр из меню системы

После создания ярлыков все игры становятся доступны в меню приложений вашего дистрибутива Linux. Запускайте игры как обычные приложения — двойным кликом или через поиск в меню.
