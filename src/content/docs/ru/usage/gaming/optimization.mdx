Запуск игр на Linux требует не только совместимости, но и правильной оптимизации для достижения максимальной производительности.

Meowrch предлагает собственное решение для автоматизации этого процесса — **MeowrchGameRun**, 
инструмент, который анализирует игры и применяет оптимальные настройки без ручного вмешательства.

В отличие от традиционных подходов с `gamemoderun` или `prime-run`, **MeowrchGameRun** использует интеллектуальный анализ исполняемых файлов, определяет используемые технологии и автоматически выбирает необходимые оптимизации для каждой конкретной игры.

## Основные возможности:

Автоматическое определение оптимизаций:
-  Анализ исполняемых файлов на наличие игровых движков (Unity, Unreal, Godot, Source)
-  Определение используемых графических API (OpenGL, Vulkan, DirectX)
-  Обнаружение поддержки DLSS и FSR
-  Анализ библиотек и DLL файлов в директории игры
-  Определение лаунчера (Steam, Lutris)

Умные оптимизации:
-  DLSS: Автоматически включается при наличии NVIDIA GPU и поддержки DLSS в игре
-  Zink: Включается для старых OpenGL игр при наличии Vulkan поддержки
-  Prime: Автоматически применяется на системах с дискретной NVIDIA видеокартой
-  GameMode: По умолчанию отключен (может навредить производительности), включается только при явном указании
-  Профили питания: Автоматически переключает на режим производительности

## Как установить?
Для установки вам потребуется скачать пакет `meowrh-tools` с помощью команды
```bash
yay -Sy meowrch-tools
```

После чего команда `meowrch-game-run` будет доступна.

## Настройка в Steam
Для конкретной игры
<Steps>
1. Откройте Steam
2. Найдите нужную игру в библиотеке
3. Кликните правой кнопкой мыши → "Свойства"
4. Во вкладке "Общие" найдите "Параметры запуска"
5. Введите `meowrch-game-run %command%`
</Steps>
