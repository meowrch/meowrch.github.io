# Редактирование конфигураций

В **meowrch** предусмотрена система **тем**. Важно понимать структуру конфигурационных файлов и правильно их редактировать.

## Структура конфигурационных файлов

<FileTree>

- ~/.config/
  - meowrch/
    - config.yaml
    - themes/
      - catppuccin-mocha/
        - waybar/
        - kitty/
        - cava/
      - your-theme/
        - waybar/
        - kitty/
        - cava/
  - waybar/ (стандартные пути - могут быть перезаписаны)
  - kitty/
  - cava/

</FileTree>

:::caution[Важное предупреждение]
Если вы вносите изменения в конфигурацию утилит по стандартным путям (`~/.config/waybar/`, `~/.config/kitty/`, `~/.config/cava/`), ваши настройки могут быть утеряны при смене темы.
:::

**Правильный способ:** редактировать файлы конфигурации в директории `.config/meowrch/themes/<название_темы>/`.

# Добавление собственных обоев

Чтобы добавить собственные обои в систему meowrch, нужно отредактировать файл конфигурации.

<Steps>

1. **Откройте файл конфигурации**
   
   Откройте основной конфигурационный файл:
   ```bash
   nano ~/.config/meowrch/config.yaml
   ```

2. **Выберите способ добавления обоев**
   
   **Для всех тем (глобально):**
   
   Найдите секцию `custom-wallpapers` и добавьте пути к вашим обоям:
   ```yaml
   custom-wallpapers:
     - /path/to/your/wallpapers/image1.jpg
     - /path/to/your/wallpapers/image2.png
   ```
   
   **Для конкретной темы:**
   
   Найдите или создайте секцию `available_wallpapers` в нужной теме:
   ```yaml
   themes:
     my-theme:
       available_wallpapers:
         - /path/to/theme/wallpapers/bg1.jpg
         - /path/to/theme/wallpapers/bg2.png
   ```

3. **Используйте маску для папок (опционально)**
   
   Для добавления всех изображений из папки используйте маску `*`:
   ```yaml
   custom-wallpapers:
     - ~/.config/meowrch/wallpapers/my-collection/*
   ```
   
   :::tip[Полезный совет]
   Маска `*` автоматически включит все файлы форматов PNG и JPG из указанной папки.
   :::

</Steps>

# Создание собственной темы

Вы можете создать собственную тему на основе существующей.

<Steps>

1. **Скопируйте существующую тему**
   
   Скопируйте одну из стандартных тем и дайте ей уникальное название:
   ```bash
   cp -r ~/.config/meowrch/themes/catppuccin-mocha ~/.config/meowrch/themes/my-custom-theme
   ```

2. **Настройте конфигурации**
   
   Отредактируйте конфигурационные файлы в скопированной теме:
   
   **Доступные конфигурации:**
   - `waybar/` - настройки панели
   - `kitty/` - настройки терминала
   - `cava/` - настройки аудиовизуализатора
   
   :::caution[Важно знать]
   - Можно изменять только существующие конфигурации
   - Не изменяйте названия конфигурационных файлов
   - Посторонние файлы будут игнорироваться
   :::

3. **Добавьте тему в конфигурацию**
   
   Откройте файл конфигурации и добавьте вашу тему:
   ```bash
   nano ~/.config/meowrch/config.yaml
   ```
   
   Добавьте следующие строки в секцию `themes`:
   ```yaml
   themes:
     my-custom-theme:
       available_wallpapers:
         - ~/.config/meowrch/wallpapers/my-theme/*
         - /path/to/your/custom/wallpapers/*
   ```

4. **Тестирование темы**
   
   Переключитесь на вашу новую тему с помощью соответствующей команды meowrch и проверьте результат.

</Steps>
