Meowrch даёт простой и быстрый способ управлять обоями вне зависимости от того, используете вы X11 или Wayland.

## Быстрая смена обоев

Основной способ — **Rofi‑селектор**:

```
~/.local/bin/rofi-menus/wallpaper-selector.sh
```

<Aside type="note" title="Самый лёгкий способ">
Использовать сочетание клавиш `Super + W`
</Aside>

Команда `set-wallpaper.sh` — это вспомогательный инструмент (если хотите применить обои напрямую по пути):

```
~/.local/bin/set-wallpaper.sh </path/to/wallpaper.jpg>
```

Он автоматически выбирает нужный инструмент:
- `feh` для **X11**
- `swww` для **Wayland**

### Повторно установить текущие обои

В системе хранится символическая ссылка на активные обои,  
что позволяет легко восстановить или применить их повторно с флагом `--current`:

```
~/.local/bin/set-wallpaper.sh --current
```

Ссылка на текущие обои:

```
~/.local/share/wallpapers/.current.wall
```

### Добавление своих обоев

Если вы хотите использовать свои обои, независимо от выбранных тем,  
просто скопируйте изображения в директорию:

```
~/.local/share/wallpapers/
```
