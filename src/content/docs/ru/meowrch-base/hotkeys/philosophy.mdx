При разработке meowrch мы стремились создать систему сочетаний клавиш, которая была бы не просто набором команд для заучивания, а логичной и интуитивной системой, основанной на ассоциациях. Наша цель — сделать так, чтобы вы **понимали** логику клавиш, а не механически их запоминали.

## Основной принцип: Super как фундамент

Клавиша **Super** (она же Windows/Command) — это центральная клавиша всей системы meowrch. Она всегда является базовой для системных действий и управления окружением.

<Aside type="tip">
**Запомните:** Super — это ваш главный инструмент взаимодействия с системой. Все основные действия начинаются с неё.
</Aside>

## Буквенные ассоциации: думай, а не запоминай

Каждая буква в сочетании с Super выбрана не случайно — она должна вызывать прямую ассоциацию с действием:

### Системные действия (Super + буква)

- **Super + A** = **A**pps (приложения) — открывает меню приложений
- **Super + W** = **W**allpaper (обои) — меняет обои рабочего стола  
- **Super + T** = **T**heme (тема) — переключает цветовую тему
- **Super + C** = **C**olor (цвет) — пипетка для распознавания цвета на экране
- **Super + L** = **L**ock (замок) — блокирует экран
- **Super + P** = **P**in (закрепить) — закрепляет окно поверх всех остальных
- **Super + N** = **N**otifications (уведомления) — история уведомлений
- **Super + B** = **B**ar (панель) — показать/скрыть панель
- **Super + E** = **E**xplorer (проводник) — открывает файловый менеджер
- **Super + Q** = **Q**uit (выход) — закрывает активное окно
- **Super + K** = **K**ill (убить) — принудительно завершает процесс окна
- **Super + X** = e**X**it (выход) — меню выключения/перезагрузки

<Aside type="note">
Обратите внимание: даже если буква не на первой позиции (как X в exit), она всё равно ярко ассоциируется с понятием на английском языке. Это помогает интернационализации клавиш.
</Aside>

## Модификаторы: уровни действий

### Super + Shift: запуск конкретных приложений

Когда вы добавляете **Shift** к Super, вы переходите на уровень запуска конкретных приложений. Здесь тоже работают ассоциации по первой букве:

- **Super + Shift + F** = **F**irefox
- **Super + Shift + C** = **C**ode (редактор кода)
- **Super + Shift + T** = **T**elegram
- **Super + Shift + O** = **O**bsidian (заметки)
- **Super + Shift + P** = **P**avucontrol (аудио микшер)
- **Super + Shift + Y** = **Y**azi (терминальный файловый менеджер)
- **Super + Shift + B** = переключение **B**ar (mewline/waybar)

<Aside type="tip">
**Логика проста:** Super выполняет системное действие, Super + Shift запускает приложение. Буква всегда намекает на название.
</Aside>

### Super + Ctrl: навигация между пространствами

Когда вы держите **Ctrl** вместе с Super, вы управляете **рабочими пространствами**:

- **Super + Ctrl + ←/→** = переключение между рабочими областями
- **Super + Ctrl + ↓** = переход на первую пустую рабочую область

**Логика:** Ctrl означает "контроль" над всем окружением, а не отдельным окном.

### Super + Shift + Ctrl: перемещение окон

Когда вы держите **все три клавиши** (Super + Shift + Ctrl), вы получаете максимальный контроль — **физическое перемещение окна** стрелками.

**Логика:** Чем больше модификаторов, тем более "тяжёлое" и точное действие вы выполняете.

## Стрелки vs цифры

### Стрелки — для фокуса и перемещения

- **Super + ←/↑/→/↓** = перемещение фокуса между окнами
- **Super + Shift + ←/↑/→/↓** = изменение размера окна
- **Super + Shift + Ctrl + ←/↑/→/↓** = перемещение окна в пространстве

### Цифры — для рабочих областей

- **Super + 1-9** = переключение на рабочую область по номеру
- **Super + Shift + 1-9** = перемещение окна на рабочую область

<Aside>
**Почему так?** Стрелки — это направления в 2D пространстве (окна на экране), цифры — это слои в 3D (рабочие области как этажи здания).
</Aside>

## Alt: альтернативные действия

**Alt** в meowrch используется для действий, которые являются "альтернативными" или "дополнительными" к основным:

- **Alt + Enter** = полноэкранный режим (альтернатива обычному размеру)
- **Alt + Tab** = переключение между окнами (классическая альтернатива фокусу стрелками)
- **Alt + Shift** = смена раскладки клавиатуры (альтернативный язык)

## Специальные клавиши: очевидные действия

Некоторые клавиши настолько очевидны, что не требуют ассоциаций:

- **PrintScreen** = скриншот области
- **Super + PrintScreen** = скриншот всего экрана  
- **Ctrl + Shift + Esc** = диспетчер задач (как в Windows)
- **Super + Enter** = терминал (универсальная конвенция в тайловых WM)
- **Super + .** = эмодзи (как в Windows и macOS)
- **Super + /** = шпаргалка по клавишам (? = помощь)

## Практика запоминания

Вместо того чтобы зубрить список, попробуйте этот подход:

1. **Подумайте о действии** — что вы хотите сделать?
2. **Переведите на английский** — как это называется?
3. **Возьмите первую букву** — она скорее всего и есть нужная клавиша
4. **Добавьте нужный уровень:**
   - Просто Super = системное действие
   - Super + Shift = запуск приложения
   - Super + Ctrl = управление пространствами
   - Super + Shift + Ctrl = перемещение окон

<Aside type="caution">
**Исключение:** Некоторые комбинации отличаются (например, Super + Space для плавающего режима), но их очень мало, и они быстро войдут в мышечную память.
</Aside>

## Кастомизация: следуйте философии

Когда вы будете добавлять свои собственные сочетания клавиш, **следуйте той же философии**:

- Используйте Super как базу для системных действий
- Выбирайте буквы, которые **ассоциируются** с действием на английском
- Используйте Shift для запуска приложений
- Используйте Ctrl для действий над всем окружением
- Не ломайте существующие паттерны — это создаст когнитивный диссонанс
