Если на X11 вы видите разрывы изображения (tearing), это типичный сценарий для NVIDIA.

## Что помогает чаще всего

Включение **ForceCompositionPipeline**.

Создайте или отредактируйте `/etc/X11/xorg.conf.d/20-nvidia.conf`:

```txt
Section "Screen"
    Identifier "Screen0"
    Option "ForceCompositionPipeline" "On"
EndSection
```

Если этого недостаточно, попробуйте:

```txt
Option "ForceFullCompositionPipeline" "On"
```

## Проверка

1. Перезагрузите X11-сессию (или ПК)
2. Проверьте, исчезли ли разрывы при прокрутке/видео/играх

<Aside type="caution" title="Минусы">
`ForceFullCompositionPipeline` может добавить задержку ввода (input lag). Если важна отзывчивость, начните с обычного `ForceCompositionPipeline`.
</Aside>
