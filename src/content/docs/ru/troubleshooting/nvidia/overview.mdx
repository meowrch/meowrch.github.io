Этот раздел посвящён самым частым проблемам NVIDIA в **Arch Linux**, включая **Wayland** и **X11**.

## Быстрая диагностика NVIDIA в Linux

Перед применением фиксов проверьте базовое состояние драйвера:

```bash
nvidia-smi
lsmod | grep -E 'nvidia|nouveau'
cat /sys/module/nvidia_drm/parameters/modeset
cat /sys/module/nvidia_drm/parameters/fbdev
systemctl status nvidia-suspend.service nvidia-resume.service nvidia-hibernate.service --no-pager
```
