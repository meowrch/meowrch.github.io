Это руководство поможет вам установить Meowrch из Windows — от создания загрузочной флешки до установки готовой системы на базе Arch Linux.

Хотите понять, что именно делает автоустановщик? См. [документацию Builder](/ru/installation/builder/). Если понадобится откатить изменения — см. [как удалить Meowrch](/ru/installation/uninstall/).

<Aside type="note" title="Что вас ждёт">
В этом гайде мы:
- Скачаем и создадим загрузочную флешку EndeavourOS
- Установим базовый Arch Linux через EndeavourOS
- Установим и настроим Meowrch поверх базовой системы
</Aside>

## 🖥️ Подготовка в Windows

### Шаг 1: Скачивание необходимых программ

<Steps>

1. **Скачайте Rufus**
   
   Перейдите на [официальный сайт Rufus](https://rufus.ie/) и скачайте последнюю версию программы.
   
   Rufus — это утилита для создания загрузочных USB-накопителей из ISO-образов.

2. **Скачайте EndeavourOS**
   
   Перейдите на [сайт EndeavourOS](https://endeavouros.com) и скачайте последний ISO-образ.
   
   EndeavourOS — это дружелюбный дистрибутив на основе Arch Linux с удобным установщиком.

3. **Подготовьте USB-накопитель**
   
   Вам понадобится USB-флешка объёмом минимум **4 ГБ**.
   
   <Aside type="caution" title="Внимание!">
   Все данные на флешке будут удалены в процессе создания загрузочного носителя.
   </Aside>

</Steps>

### Шаг 2: Создание загрузочной флешки

<Steps>

1. **Запустите Rufus**
   
   Откройте скачанный файл Rufus. При необходимости разрешите программе внести изменения в систему.

2. **Выберите USB-устройство**
   
   В поле "Устройство" выберите вашу флешку. Rufus автоматически определит подключенные накопители.

3. **Выберите ISO-образ**
   
   - Нажмите "ВЫБРАТЬ" рядом с полем "Метод загрузки"
   - Найдите скачанный ISO-файл EndeavourOS и выберите его

4. **Настройте параметры**
   
   - **Схема раздела**: GPT (для современных компьютеров с UEFI)
   - **Целевая система**: UEFI (не CSM)
   - Остальные настройки можно оставить по умолчанию

5. **Создайте загрузочную флешку**
   
   Нажмите "СТАРТ" и дождитесь завершения процесса (обычно 5-15 минут).

</Steps>

## 💾 Установка базовой системы

### Шаг 3: Подготовка места на диске в Windows

Перед установкой рекомендуем заранее выделить свободное место под Meowrch в Windows:

<Steps>

1. **Откройте Управление дисками**
   
   Нажмите Win + X → Управление дисками (Disk Management).

2. **Выберите основной раздел**
   
   Найдите самый большой раздел (обычно диск C:), кликните правой кнопкой → "Сжать том" (Shrink Volume).

3. **Укажите объём для сжатия**
   
   Введите количество гигабайт, которое хотите выделить под Meowrch (рекомендуется от 40 ГБ и больше), нажмите "Сжать".

4. **Проверьте результат**
   
   Должен появиться блок "Неразмеченное пространство" (Unallocated) соответствующего объёма — оставьте его неразмеченным, ничего не создавайте.

</Steps>

### Шаг 4: Загрузка с флешки

<Steps>

1. **Перезагрузите компьютер**
   
   Не извлекая флешку, перезагрузите компьютер.

2. **Войдите в меню загрузки**
   
   При запуске нажмите клавишу для вызова Boot Menu:
   - **Asus**: F8 или Esc
   - **MSI**: F11
   - **Gigabyte**: F12
   - **ASRock**: F11
   - **Dell**: F12
   - **HP**: F9
   - **Lenovo**: F12

3. **Выберите USB-устройство**
   
   В меню загрузки выберите вашу флешку и нажмите Enter.

4. **Загрузитесь в EndeavourOS**
   
   Выберите первый пункт в меню EndeavourOS и нажмите Enter.

</Steps>

### Шаг 5: Установка через EndeavourOS

<Steps>

1. **Запустите установщик**
   
   В загруженной системе нажмите "Start the installer" в приветственном окне.

2. **Выберите тип установки**
   
   Выберите "Online" для доступа ко всем окружениям рабочего стола.

3. **Настройте локализацию**

4. **Выберите окружение рабочего стола**
   
   <Aside type="tip" title="Важно для Meowrch!">
   Выберите **"No Desktop"** — это критично важно!
   
   Meowrch установит свои оконные менеджеры (BSPWM/Hyprland) и не требует предустановленного окружения.
   </Aside>

5. **Выберите загрузчик**
   
   <Aside type="caution" title="Обязательно!">
   Выберите **GRUB** в качестве загрузчика!
   
   Это необходимо для корректной работы Meowrch и мультизагрузки с Windows.
   </Aside>

6. **Разметка диска**
   
   Выберите **Заменить раздел** (Replace a partition) и найдите **свободное неразмеченное пространство**, которое вы подготовили в Windows.
   Выберите его.


7. **Создайте пользователя**
   
   - Введите ваше имя и имя пользователя
   - Создайте надёжный пароль

8. **Завершите установку**
   
   Нажмите "Install" и дождитесь завершения установки (обычно 15-30 минут).

9. **Перезагрузитесь**
   
   После завершения установки перезагрузите компьютер и извлеките флешку.

</Steps>

## 🐱 Установка Meowrch

### Шаг 6: Первичный вход

При варианте **No Desktop** система загрузится **сразу в TTY** (полноэкранный текстовый терминал).

<Steps>

1. **Войдите в систему**
   
   Введите ваш логин и пароль, созданные при установке EndeavourOS.

2. **Обновите систему**
   
   Выполните:
   ```bash
   sudo pacman -Syu
   ```

3. **Установите Git**
   
   ```bash
   sudo pacman -S git
   ```

</Steps>

### Шаг 7: Установка Meowrch

<Steps>

1. **Клонируйте репозиторий Meowrch**
   
   ```bash
   git clone --recurse-submodules --depth 1 https://github.com/meowrch/meowrch
   ```

2. **Перейдите в каталог проекта**
   
   ```bash
   cd meowrch
   ```

3. **Запустите автоматическую установку**
   
   ```bash
   sh install.sh
   ```

4. **Следуйте инструкциям установщика**
   
   <Aside type="note" title="Навигация в установщике">
   - **Выбор пункта**: Пробел
   - **Переход к следующему шагу**: Enter
   - **Выход**: Ctrl + C
   </Aside>

5. **Перезагрузите систему**
   
   <Aside type="caution" title="Обязательная перезагрузка!">
   После установки **необходимо** перезагрузиться для применения всех изменений.
   </Aside>
   
   ```bash
   sudo reboot
   ```

</Steps>

## 🔧 Решение возможных проблем

### Проблема: Не загружается с флешки

- Убедитесь, что Secure Boot отключён в BIOS
- Попробуйте другой USB-порт
- Пересоздайте загрузочную флешку

### Проблема: Ошибки при установке EndeavourOS

- Проверьте подключение к интернету
- Убедитесь, что диск не повреждён
- Попробуйте offline-установку

### Проблема: Meowrch не устанавливается

- Убедитесь, что система обновлена: `sudo pacman -Syu`
- Проверьте подключение к интернету
- Запустите установку от имени пользователя (не root)
