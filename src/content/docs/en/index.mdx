:::note[The Problem with Ready-Made Rice Configs]
**Most ready-made rice setups for Arch solve only one problem:**

- üé® Either **beauty**, but the system eats RAM.
- ‚ö° Or **performance**, but the UI looks ugly.
- üõ†Ô∏è Or **functionality**, but the code turns into "spaghetti" that you can't fix yourself.
:::

## Meowrch is a different approach

We don't sacrifice one for the other. Instead, we create a system where each component is chosen based on the principle of **"maximum results with minimum resources"**.

**But most importantly** ‚Äî we don't limit ourselves to beautiful configs.

<TerminalBlock>
```bash
# 1. Clone the repository
git clone https://github.com/meowrch/meowrch --depth 1 --single-branch
cd meowrch

# 2. Run the installer
sh install.sh

# 3. Reboot
reboot
```
