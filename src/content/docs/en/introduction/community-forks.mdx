Meowrch is an open-source project. The community sometimes ships **alternative builds and ports** for other platforms and distributions.
These are **not official Meowrch releases** — each build is maintained and supported by its maintainer.

## What counts as a “port” or “build”

This section includes:
- **Ports to other distributions** (e.g., NixOS);
- **Builds for other platforms** (e.g., MacBook on Apple Silicon).

## Known community builds

- **Meowrch for NixOS** — a full NixOS port (home-manager + flakes).  
  Repository: [NixOS-Meowrch](https://github.com/Redm00use/NixOS-Meowrch)
- **Meowrch for MacBook (Apple Silicon)** — Arch-based build for MacBook M1 via Asahi Linux.  
  Repository: [meowrch-asahi](https://github.com/Redm00use/meowrch-asahi)
- **Meowrch for MacBook + NixOS** — NixOS on MacBook M1 via Asahi Linux.  
  Repository: [NixOS-Asahi-Kennel](https://github.com/Redm00use/NixOS-Asahi-Kennel)

## Before you install

- These builds **may differ** in functionality from the main Meowrch release.
- Updates depend on the fork maintainer.
- Installation and support are usually handled **via the fork repository**, not the main Meowrch repo.

If you need a stable and fully supported version, use the **main Meowrch for Arch**.

:::caution[Important]
These are **unofficial community ports**. The repositories are maintained by enthusiasts and are not part of the official Meowrch codebase.  
We’re not responsible for their operation, but we welcome community contributions to the ecosystem.
:::

:::note[Want to add your fork?]
If you maintain a build, open an issue or PR and include the repository link, installation instructions, and support status.
:::
