# Configuration Editing

**meowrch** provides a **themes** system. It's important to understand the configuration file structure and edit them properly.

## Configuration File Structure

<FileTree>
  * \~/.config/
    * meowrch/
      * config.yaml
      * themes/
        * catppuccin-mocha/
          * waybar/
          * kitty/
          * cava/
        * your-theme/
          * waybar/
          * kitty/
          * cava/
    * waybar/ (default paths - may be overwritten)
    * kitty/
    * cava/
</FileTree>

:::caution[Important Warning]
If you make changes to utilities configuration via default paths (`~/.config/waybar/`, `~/.config/kitty/`, `~/.config/cava/`), your settings may be lost when you change themes.
:::

**Correct way:** edit configuration files in the `.config/meowrch/themes/<theme_name>/` directory.

# Adding Custom Wallpapers

To add your own wallpapers to the meowrch system, you need to edit the configuration file.

<Steps>
  1. **Open the configuration file**

     Open the main configuration file:

     ```bash
     nano ~/.config/meowrch/config.yaml
     ```

  2. **Choose how to add wallpapers**

     **For all themes (globally):**

     Find the `custom-wallpapers` section and add paths to your wallpapers:

     ```yaml
     custom-wallpapers:
       - /path/to/your/wallpapers/image1.jpg
       - /path/to/your/wallpapers/image2.png
     ```

     **For a specific theme:**

     Find or create an `available_wallpapers` section in the desired theme:

     ```yaml
     themes:
       my-theme:
         available_wallpapers:
           - /path/to/theme/wallpapers/bg1.jpg
           - /path/to/theme/wallpapers/bg2.png
     ```

  3. **Use folder masks (optional)**

     To add all images from a folder, use the `*` mask:

     ```yaml
     custom-wallpapers:
       - ~/.config/meowrch/wallpapers/my-collection/*
     ```

     :::tip[Helpful hint]
     The `*` mask will automatically include all PNG and JPG files from the specified folder.
     :::
</Steps>

# Creating Your Own Theme

You can create your own theme based on an existing one.

<Steps>
  1. **Copy an existing theme**

     Copy one of the standard themes and give it a unique name:

     ```bash
     cp -r ~/.config/meowrch/themes/catppuccin-mocha ~/.config/meowrch/themes/my-custom-theme
     ```

  2. **Configure the theme**

     Edit the configuration files in the copied theme:

     **Available configurations:**

     * `waybar/` - panel settings
     * `kitty/` - terminal settings
     * `cava/` - audio visualizer settings

     :::caution[Important to Know]
     - Only existing configurations can be modified
     - Don't change configuration file names
     - Extraneous files will be ignored
     :::

  3. **Add the theme to configuration**

     Open the configuration file and add your theme:

     ```bash
     nano ~/.config/meowrch/config.yaml
     ```

     Add the following lines to the `themes` section:

     ```yaml
     themes:
       my-custom-theme:
         available_wallpapers:
           - ~/.config/meowrch/wallpapers/my-theme/*
           - /path/to/your/custom/wallpapers/*
     ```

  4. **Test the theme**

     Switch to your new theme using the appropriate meowrch command and check the result.
</Steps>
