üîß Patch System

Section titled ‚Äúüîß Patch System‚Äù

Pawlette supports two modes of configuration application:

Full Replacement

Section titled ‚ÄúFull Replacement‚Äù

Regular configuration files will completely replace existing settings.

Partial Changes (Patches)

Section titled ‚ÄúPartial Changes (Patches)‚Äù

Patch suffixes

  • .prepaw ‚Äî content is added before the original configuration
  • .postpaw ‚Äî content is added after the original configuration
  • .jsonpaw ‚Äî JSON object is merged into the target JSON file (deep merge; applied before .prepaw/.postpaw)

JSON merge (.jsonpaw)

Section titled ‚ÄúJSON merge (.jsonpaw)‚Äù

If the target file is JSON, you can place a file next to it with the .jsonpaw suffix (e.g. waybar/config.json.jsonpaw). Its content must be a JSON object; it will be merged into the target file:

  • existing keys are overridden by .jsonpaw values;
  • missing keys are created;
  • nested objects are merged recursively.
